# AQUAZONE Water Park 2030

**Web premium futurista 2030** totalmente funcional y lista para producción.  
Desarrollada con Next.js, React, TypeScript y Framer Motion.

---

## Características principales

- Home con video de fondo, animaciones 3D y botones flotantes.  
- Categorías:  
  - **Parques Flotantes**: Piscinas y playas modulares.  
  - **Parques Urbanos/Terrestres**: 5 actividades + Kamikaze Jump.  
- Galerías interactivas 3D: Urbano, Flotante, Terrestre y Kamikaze.  
- Botón flotante WhatsApp y botones de redes sociales.  
- Formularios de contacto y presupuesto, integrados con Hotmail SMTP.  
- Panel administrativo privado: Calendario, Contactos, Presupuestos y Campañas.  
- IA integrada (OpenAI) para generación de contenido, marketing y respuestas automáticas.  
- Preparado para futuras integraciones: DALL·E, Sora y automatización de redes sociales.  
- Adaptado a móvil, tablet y PC.  
- Diseño avanzado y coherente 2030.

---

## Tecnologías

- Next.js 14 (App Router)  
- React 18 + TypeScript  
- Framer Motion  
- React Icons  
- Date-fns  
- OpenAI API / DALL·E / Sora  
- SMTP Hotmail  
- Vercel para despliegue
